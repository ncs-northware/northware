# BrandIcon

Die BrandIcon-Komponente zeigt das Icon der mit dem Prop `service` definierten Northware App. 
Basierend auf dem `service` Prop werden die SVG-Icons `IconCockpit`, `IconFinance` oder `IconTrader` angezeigt.
<ViewSource path="components/base/brand.tsx" />

<Canvas of={BrandIconStories.Cockpit} />
<Controls of={BrandIconStories.Cockpit} />

## Benutzung

```tsx
import { BrandIcon } from '@northware/ui/components/brand';

// ein festgelegter ServiceType
export default function Example() {
    return (
        <BrandIcon service="cockpit"/>
    )
}

// ein geerberter ServiceType
export default function Example({service} : {service: ServiceType}) {
    return (
        <BrandIcon service={service}/>
    )
}
```
