# DataTable

Die Komponente `DataTable` ist eine komplexe Tabelle die die Komponente `Table` mit [@tanstack/react-table](https://tanstack.com/table/v8) erweitert.
Die Tabelle lässt sich sortieren und filtern und enthält eine pagination.
