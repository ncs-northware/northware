# Card

Eine Card ist eine Komponente, mit der verschiede Inhalte dargestellt werden.
Die Card kann einen CardTitle haben. Innerhalb von CardContent  kann ein beliebiger Inhalt dargestellt werden.

## Benutzung

```tsx
import {
  Card,
  CardContent,
  CardDescription,
  CardFooter,
  CardHeader,
  CardTitle,
} from "@northware/ui/components/panels/card"

<Card>
  <CardHeader>
    <CardTitle>Card Title</CardTitle>
    <CardDescription>Card Description</CardDescription>
  </CardHeader>
  <CardContent>
    <p>Card Content</p>
  </CardContent>
  <CardFooter>
    <p>Card Footer</p>
  </CardFooter>
</Card>
```
