# PasswordInput

Ein Passwort Eingabefeld mit integriertem Button um die Sichtbarkeit zu kontrollieren.

<ViewSource path="components/password-input.tsx"/>

<Canvas of={PasswordInputStories.Default} />

Die Komponente zeigt ein Input-Feld vom Typ `password`. 
Durch einen Klick auf den intigrierten Button kann der Typ auf `text` und 
wieder zurück geändert werden. So kann der Nutzer sich den eingegebenen Wert bei Bedarf ansehen.
