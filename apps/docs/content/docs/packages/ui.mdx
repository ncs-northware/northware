---
title: "@northware/ui"
description: Styling, Component Library und Grundstruktur für alle Northware Apps
---

`@northware/ui` ist die Component Library für die Northware Apps. Sie enthält grundlegende aber auch komplexe Komponenten, 
die innerhalb aller Northware Apps genutzt werden können. Das Package stellt auch das CSS-Styling, Helper-Funktionen, 
Icons und Hooks bereit. So muss innerhalb der Apps nur noch wenig über die UI nachgedacht werden.

<Callout type="info">
Als Dokumentation von `@northware/ui` wird das [Northware Storybook](https://northware-storybook.vercel.app/) verwendet.
</Callout>
